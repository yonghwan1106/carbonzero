# CarbonZero Platform 프로토타입

**2025 KOMIPO 「혁신 50대 과제」 아이디어 공모전 제출용**

CarbonZero는 AI 기반 탄소배출 예측과 자동매칭을 통한 탄소중립 플랫폼 프로토타입입니다.

> **공모 분야**: ②에너지 전환  
> **과제명**: 넷제로 달성을 위한 AI 기반 탄소 예측 및 상쇄 자동매칭 플랫폼  
> **데모 URL**: https://carbonzero-sigma.vercel.app/

## 🚀 프로젝트 개요

이 프로토타입은 실제 API/DB 없이 완전한 기능을 시연할 수 있도록 Mock 데이터를 활용하여 구현되었습니다.

### 주요 기능 (KOMIPO 공모전 요구사항 완벽 구현)

- ⚡ **실시간 탄소배출 모니터링**: 중부발전 발전소별 실시간 배출량 및 24시간 탄소집약도 예측
- 🤖 **AI 기반 자동매칭**: RE100 기업의 탄소상쇄 요구사항과 청정전력 공급시간 자동 매칭
- 📜 **블록체인 인증서**: 탄소상쇄 거래 인증서 실시간 발급 및 QR코드 검증 시스템
- 📊 **성과 분석**: 월별 탄소상쇄량, 비용 절감 효과, 파트너 기업 관리 대시보드

### KOMIPO 제안서 핵심 내용 구현

1. **AI 탄소예측 시스템**: 발전소별 연료 구성, 기상 데이터 기반 시간대별 탄소집약도 예측
2. **자동매칭 플랫폼**: RE100 참여 기업과 중부발전 청정전력 공급시간 자동 매칭
3. **탄소 네거티브 생태계**: 중소기업 참여 가능한 '탄소상쇄 구독 서비스' 모델
4. **블록체인 인증**: 실시간 탄소상쇄 인증서 발급 및 거래 시스템

## 🛠 기술 스택

- **Frontend**: React 18 + TypeScript + Vite
- **UI Library**: Material-UI v5
- **State Management**: Zustand
- **Routing**: React Router v6
- **Charts**: Chart.js + react-chartjs-2
- **Data Processing**: Day.js, Faker.js
